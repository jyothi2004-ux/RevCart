   - Each person practices their section

2. **Print Backups**
   - Aman: Print greeting, tech stack, Angular technologies, conclusion
   - Dinesh: Print CartService explanation
   - Jyothi: Print UI testing notes
   - Akshita: Print edge cases notes

3. **Have Live Demo Ready**
   - `cd Frontend && npm start`
   - Opens to localhost:4200
   - Add items to cart to show it working
   - Show the calculations updating

4. **Prepare Optional Demo:**
   - Show live application working
   - Browser with localhost:4200 open
   - Click through a purchase flow
   - Show admin dashboard (if time allows)
